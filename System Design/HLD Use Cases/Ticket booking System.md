It's a online platform that allows user to purchase tickets of the movies, events etc

1. Functional Requirement 
	- user should be able to search and event based on some title, location and date 
	- user should be able to view the details of the events (seat, descriptions, metadata)
	- user should be able to do booking for that event 
	
2. Non Functional Requirement 
	- Scale: 100 DAU 
	- CAP Theorem : Highly available with respecting to searching and view events and highly consistent with respect to booking a particular ticket 
	
3. Identify Core Entity 
	- User 
	- Event (Movie / Concert )
	- Venue (Hall / Location)
	- Ticket
