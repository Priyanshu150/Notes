
System design should depend on following patterns :- 

1. Traffic Pattern (Constant v/s Bursty)
2. Latency Tolerance 
3. DevOps team size 
4. Cost constraints 
5. Vendor lock-in concerns 
6. Execution Time and Performance 
7. Stateful v/s Stateless
8. Control and Customization
9. Development and Deployment speed 
10. Tooling / Ecosystem
