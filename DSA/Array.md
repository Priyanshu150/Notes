Ways to solve array problems :- 

1. Subarray problems :- 
	1.1. Prefix - Sum 
		Calculate the sum of array and store it. Use the sum value to calculate sum of all the subarray  optimally. 
	
	1.2. Hashing 
		Hash the value of a subarray in order to use it in future and according to the need 
	
	 1.3. Sliding window
		Apply two pointer technique to create a window and slide the window according to the requirement
